# 26_stubby

Code and manuscript for "Mount Fuji's stubby peak: the genotypic density of additive landscapes near maximal fitness."

## Repository structure

- `src/` — Python code for fitness landscape computations, saddle-point approximations, and tail counting
- `figs/` — Jupyter notebooks that generate all paper figures (`fig1/`–`fig5/`), plus shared styling (`style_config.py`)
- `data/` — precomputed data files used by the notebooks
- `manuscript/` — compiled paper (`main.pdf`) and supplemental information (`si.pdf`)

## Setup

This project uses [uv](https://docs.astral.sh/uv/) for dependency management.

```bash
# Install dependencies and create the virtual environment
uv sync

# Install with dev dependencies (Jupyter, nbconvert)
uv sync --group dev
```

## Generating figures

Each figure is generated by the corresponding notebook in `figs/`:

```bash
uv run jupyter nbconvert --to notebook --execute figs/fig1/fig1.ipynb
```
